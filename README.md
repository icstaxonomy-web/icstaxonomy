# ICS Safety and Security Awareness Taxonomy Resources

This repository contains supporting resources for the research work introducing the **ICS Safety and Security Awareness Taxonomy**, a hierarchical framework designed to structure and assess safety and cybersecurity awareness in Industrial Control Systems (ICS) environments.

The taxonomy integrates safety and cybersecurity perspectives through a human-centered Information Security Awareness (ISA) approach based on **Knowledge, Attitude, and Behavior (KAB)** components.

## Repository Contents

- **ISA Taxonomy for ICS.xlsx**  
  The complete taxonomy structure, including:
  - Focus Areas
  - Sub-Focus Areas
  - Topics
  - Associated KAB criteria

- **links**  
  Supporting references and links related to the taxonomy and its construction.

## Purpose

These resources are provided to support **transparency, reproducibility, and further research** on safety–security awareness in ICS and cyber–physical systems.
